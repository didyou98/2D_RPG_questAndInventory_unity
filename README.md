# 2D_RPG_questAndInventory_unity
Unity를 이용한 2D게임 퀘스트, 대화 및 인벤토리창 구현
## 형식
- Top-Down View

## 주요기능
1. NPC와 대화 기능
2. NPC에게서 퀘스트 수주
3. 인벤토리 기능
4. 아이템 획득 기능
